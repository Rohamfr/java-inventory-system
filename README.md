# Java Inventory System

## Overview
A Java-based inventory management system designed to manage game-related inventory data using object-oriented programming principles.

## Features
- Add and manage inventory items
- Inventory persistence and data saving
- Unit testing with JUnit
- Modular project structure using Maven
- Exception and logic handling

## Technologies
- Java
- Maven
- JUnit

## Project Structure
- `InventorySystem.java` — main inventory logic
- `Game.java` — game-related entity/model
- `GameTest.java` — tests for game logic
- `InventoryLogicTest.java` — inventory functionality tests
- `InventoryPersistenceTest.java` — persistence and storage tests

## Concepts Used
- Object-Oriented Programming (OOP)
- Unit Testing
- File Persistence
- Modular Java Project Structure
